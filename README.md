## Features

- **Group your tools** — organize apps, files, folders, and URLs into named groups with custom emoji icons
- **One-click launch** — open anything instantly from the widget bar
- **Always there** — sits on your desktop, out of the way but always accessible. Drag it anywhere.
- **URLs & Bookmarks** — add websites and browser bookmarks alongside your apps
- **Windows Apps** — full support for modern Windows Store apps, not just .exe files
- **Customizable** — pick any emoji as a group icon, name groups however you want

## Installation

1. Download the MSI installer from the [releases page](https://github.com/Doogals/AppLauncher/releases)
2. Right-click the downloaded file → **Properties** → check **Unblock** → OK
3. Run the installer

> Note: Windows SmartScreen may show a warning since the app is not yet code-signed. Click "More info" → "Run anyway", or unblock via Properties as above.

## Development

**Stack:** Tauri v2 + Vite, Rust, Vanilla JS/HTML/CSS

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build installer
npm run tauri build
```
